# AGENTS.md

This folder has one agent rule: the two parity mapping trees are generated and must be regenerated, not hand-maintained.

## Generated Parity Mapping Folders (docs/parity)

- `docs/parity/source-mapping/` and `docs/parity/test-mapping/` are generated by the Rust crate:
  - `crates/parity-cli`
- Regenerate with:
  - `cargo run -p parity-cli -- source`
  - `cargo run -p parity-cli -- test`
  - `cargo run -p parity-cli` (both)

Do not directly edit any files under:

- `docs/parity/source-mapping/`
- `docs/parity/test-mapping/`

Update those folders only by running the generator commands above.
